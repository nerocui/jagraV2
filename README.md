# jagraV2
A task management service for any organization.

## Tech Stack
- Backend: ASP.NET Core 3.0
- Database: SQL Server
- ORM: Entity Framework Core 3.0
- Frontend: React, Typescript
- Native App: UWP

## Features
- Multi Tenancy
- Organization/Task Management
- Dark Mode
